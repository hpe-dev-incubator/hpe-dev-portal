## Automate what happens next using HPE OpsRamp Process Automation

May 20, 2026

Join our next HPE Developer Community Meetup to learn more about HPE OpsRamp Process Automation.
Process Automation enables you to set up and automatically run tasks related to platform events in the system. Use it to trigger workflows on signals, timers, or events to save time and reduce manual operations. It can also be used as a scheduler platform for customers who need to configure tasks that run on various schedules.
In the session, you’ll have the opportunity to understand HPE OpsRamp Process Automation implementation and usecase walkthrough. We look forward to seeing you there!
