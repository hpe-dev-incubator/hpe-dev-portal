KubeDirector: The easy way to run complex, stateful applications on Kubernetes
KubeDirector is an open source Apache project, initiated by BlueData, designed to make it easy to run complex, stateful, scale-out application clusters on Kubernetes. With KubeDirector, you can run clusters for AI, machine learning, and big data analytics on Kubernetes without writing a single line of Go code. Learn more here.

Deploying complex, stateful applications on Kubernetes with KubeDirector
This BlueData webinar provides an overview of the KubeDirector architecture, shows how to author the metadata and artifacts required for a typical stateful application (e.g. Spark, Jupyter, and Cassandra), and demonstrates the deployment and management of the cluster on Kubernetes using KubeDirector.
